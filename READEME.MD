# Project1
## How to use

Compile and execute

```
javac -d out src/*.java
java -cp out Main
```

First select .vert files

### Commands
The following keyboard shortcuts are available during execution of the program:
- **Space** Start or stop the curvature flow animation.

- **T** Toggle the display of tangent vectors (red arrows).

- **N** Toggle the display of normal vectors (blue arrows).

- **K** Toggle the display of curvature vectors (green arrows).

- **P** Toggle the display of vertices (points).

- **A** Toggle all visual elements (tangents, normals, curvature, and points) on or off at once.
